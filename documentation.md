# Documentation

## Module

It is defined on ``package.json`` or as an attribute when JS file is included in HTML.
```json
{
  "...": {},
  "type": "commonjs"
}
```

```json
{
  "...": {},
  "type": "module"
}
```

```html
<script src="common_file.js" type="commonjs"></script>
<script src="module_file.js" type="module"></script>
```
